Return to the Tomb of Horrors

This page contains the campaign chronicle of a Rolemaster game run by Dave and Helen Brubeck. The campaign was set in the world of Greyhawk from TSR (now Wizards of the Coast, owned by Hasbro I think), though the RMSS rules are used. This was a high level campaign; the characters began at 20th level, and at the conclusion had reached 22nd level. This game is now concluded, the adventurers having successfully thwarted the machinations of the dread demilich Acerak.

Other Passages

The same gaming group previously played in another campaign, set in the Shadow World of Kulthea. That game is now in permanent hibernation, though the Chronicles from it remain online.

The group is now playing in a new campaign set in the world of Greyhawk.
