Interesting links

If you enjoy this site, I am gratified. I'll also suggest a few other sites for your persusal:
  • OOTS! The Order of the Stick webcomic is a favorite of all of the players in this gaming group.
  • Edward Keyes wrote a series of tales of Korel the Necromancer. Korel was a character in his AD&D campaign. Some of the stories are from their gaming sessions and some are original fantasy. "Patientia Est Virtus" in particular is excellent.
  • And then there is the grand-daddy of them all, The Adventurers. Thomas Miller has been writing the continuous story of an AD&D campaign since 1992. The body of work is quite extensive, and makes an interesting read if you've got a couple months of free time.

Helper Software

You'll need a few plug-ins and helper applications to get the full experience from this site.
There are a number of images scattered throughout the Chronicle, illustrating scenes from the adventures. Being set in dungeons and other dim and scary places, these images tend to be quite dark. Unfortunately, "dark" on one monitor is "mid-grey" on another, making it quite difficult to generate images which look good on everyone's screen.
This is where you can help. It is fairly simple to turn on color correction on your computer, to try to make different monitors correspond more closely. On a Macintosh this is done using ColorSync. On Windows, look for an item named "Gamma Correction" in the Advanced section of the Displays control panel; it should be set to about 1.8.
Movies on the site are in Quicktime format.

 

About these pages...

These pages are produced on a Powerbook G4.
HTML pages are created using SubEthaEdit, a truly awesome network-shareable text editor. I used to be a BBEdit fan, but now use SubEthaEdit exclusively (even when I'm not using the network-aware features).
Some of the early 3D images were rendered using POV-Ray, an incredible free raytracer.
  Character (and monster) images were created using Poser, using many 3D models from DAZ Productions.
  Some images in the Chronicle were rendered using Bryce, now a product of DAZ 3D.
  Cinema 4D XL was also used for some of the later images.
